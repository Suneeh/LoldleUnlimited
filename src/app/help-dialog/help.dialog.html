How to play?

Guess the champion from Riot's Game "League of Legends"! After you guessed a Champion you will get information about the release year, gender, range type, the champions used resource type, their main position(s), the region they come from as well as the species they belong to.

Green indicates an exact match.

Orange indicates partial match.

Red indicates no match at all.

northsouth indicates if the answer is above or below your guess.

Properties

These are the details for each piece of information you will get:

Release year

When the champion was released to the live servers.

Possible values: Any year between 2009 and {{date}}.

Gender

Possible values: Male, Female and Other.

Range type

How the champion hits other champions in game (Auto Attack). Some champions might have multiple true values.

Possible values: Meele and Ranged

Resource

The type of resource the champion uses to cast spells in the game.

Possible values: Mana, Energy, Manaless, ...

Position

The most played position(s) for the champion in the game. Some champions might have multiple true values.

Possible values: Mana, Energy, Manaless, ...

Region

Where the champion comes from or resides according to lore. Some champions might have multiple true values.

Possible values: Ionia, Shadow Isles, Targon, ...

Species

The race of the champion. Some champions might have multiple true values.

Possible values: Human, Vastayan, Yordle, ...
